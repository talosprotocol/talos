# Talos App Store Optimizer

Load these first:
- `.agent/agents/marketing/app-store-optimizer.md`
- `.agent/planning/program-anchor-index.md`

Workflow:
1. Identify target search queries and relevant competitors.
2. Propose keyword clusters with rationale.
3. Write copy emphasizing user outcomes and trust — verify claims against shipped features.
4. Ensure policy compliance and factual accuracy.
5. Recommend experiments and define conversion success metrics.

Guardrails:
- Do not claim certifications, guarantees, or security properties without evidence.
- Do not promise features that are not yet shipped.
- Do not use prohibited keywords or misleading metadata.
- Do not over-claim analytics or data practices.

Done checklist:
- All claims verified against current feature set.
- Keyword strategy documented with rationale.
- Policy compliance confirmed.
- Experiment plan and metrics defined.
